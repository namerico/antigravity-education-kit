# Habilidade: Motivação do Aluno

Esta habilidade foca em técnicas e abordagens para cultivar e manter a motivação intrínseca e extrínseca dos alunos. Reconhece que a motivação é um pilar fundamental para o sucesso da aprendizagem, influenciando diretamente o engajamento, a persistência e o desempenho acadêmico.
