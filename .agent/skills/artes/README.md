# Habilidade: Artes

Esta habilidade abrange o ensino das diversas formas de expressão artística, incluindo artes visuais, música, dança e teatro. O objetivo é desenvolver a sensibilidade estética, a criatividade, a capacidade de expressão e a compreensão das manifestações culturais e históricas da arte.
