# Habilidade: Ciências Humanas

Esta habilidade abrange o ensino de História, Geografia, Sociologia e Filosofia, essenciais para a compreensão das complexas relações humanas, o desenvolvimento da cidadania e a formação de um pensamento crítico e reflexivo sobre o mundo. O objetivo é capacitar os alunos a analisar eventos, processos e estruturas sociais, culturais e políticas.
