# Habilidade: Avaliação Formativa

Esta habilidade concentra-se na avaliação como um processo contínuo e integrado ao ensino, com o objetivo principal de apoiar e melhorar a aprendizagem do aluno. Diferente da avaliação somativa, que foca no resultado final, a avaliação formativa é um guia para o percurso.
