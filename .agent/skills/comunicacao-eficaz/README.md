# Habilidade: Comunicação Eficaz

Esta habilidade é essencial para qualquer agente educacional, pois a transmissão clara de informações e a capacidade de ouvir ativamente são a base para um processo de ensino-aprendizagem bem-sucedido. Foca em técnicas que garantem que a mensagem seja recebida e compreendida como pretendido.
