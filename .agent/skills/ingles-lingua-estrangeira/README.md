# Habilidade: Inglês - Língua Estrangeira

Esta habilidade foca no ensino da língua inglesa como um meio de comunicação global. O objetivo é desenvolver as quatro habilidades linguísticas (leitura, escrita, fala e escuta), além de promover a compreensão cultural e a fluência em diferentes contextos.
