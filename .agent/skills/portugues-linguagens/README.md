# Habilidade: Português e Linguagens

Esta habilidade abrange o ensino da língua portuguesa e suas diversas manifestações, desde a gramática normativa até a análise literária e a produção textual. O objetivo é desenvolver a competência comunicativa dos alunos em diferentes contextos e gêneros.
